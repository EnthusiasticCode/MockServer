Class Methods

commandArrayFromObjectArray:

Given an array of objects, return an array of commands.

+ (NSArray *)commandArrayFromObjectArray:(NSArray *)array

Parameters

array

The input array

Return Value

An array guaranteed to contain only KMSCommand objects.

Discussion

This routine is primarily used for converting arrays of objects loaded from disc.

Any command object will just be returned untouched. NSData objects are converted into KMSSendDataCommand objects. NSNumber objects are converted into KMSPauseCommand objects. NSString objects are generally converted into either KMSSendStringCommands. A few specific string values are converted into other command objects.
